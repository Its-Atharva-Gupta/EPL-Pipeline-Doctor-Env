UNDERSTAND CURRICULUM LEARNING
───────────────────────────────

Each episode:
  1. Curriculum picks fault type (biased toward weak spots)
  2. Fault is injected into warehouse
  3. Agent gets alert + available tools
  4. Agent takes steps and gets rewards
  5. At episode end: record_outcome() updates EMA mastery

Mastery EMA evolution:
  new_ema = 0.8 × old_ema + 0.2 × success
  (exponential moving average, window ~10 episodes)

Tier progression:
  ema ≥ 0.7 → promote to harder tier
  ema ≤ 0.3 → demote to easier tier

Over 20 episodes:
  Ep 1-5:   All warmup tier, low difficulty
  Ep 5-10:  Learned faults promote to beginner
  Ep 10-15: Weak spots targeted by adversarial designer
  Ep 15-20: Curriculum focuses on hardest faults

See this in action:
  $ python3 train_traced.py --episodes 20 --dry-run
